Liste și Dicționare în Python

⏱ ~25 minute · Nivel: Începător

Listele și dicționarele sunt structurile de date fundamentale în Python. Înțelegerea lor profundă îți va permite să scrii cod eficient și elegant.

Ce este o Listă?

O listă este o colecție ordonată și mutabilă de elemente. Poate conține orice tip de date — numere, șiruri, alte liste.

python
# Creare listă fructe = ["măr", "banană", "portocală"] numere = [1, 2, 3, 4, 5] mixt = ["text", 42, True, 3.14] # Accesare elemente (indexare de la 0) print(fructe[0]) # "măr" print(fructe[-1]) # "portocală" (ultimul) # Operații comune fructe.append("struguri") # adaugă la final fructe.insert(1, "kiwi") # inserează la poziție fructe.remove("banană") # șterge element lungime = len(fructe) # numărul de elemente

Ce este un Dicționar?

Un dicționar stochează perechi cheie: valoare. Este structura perfectă pentru a reprezenta obiecte cu proprietăți.

python
# Creare dicționar student = { "nume": "Maria Ionescu", "vârstă": 22, "note": [9, 8, 10], "activ": True } # Accesare valori print(student["nume"]) # "Maria Ionescu" print(student.get("email", "N/A")) # "N/A" (valoare default) # Modificare și adăugare student["email"] = "maria@exemplu.ro" student["vârstă"] = 23 # Iterare for cheie, valoare in student.items(): print(f"{cheie}: {valoare}")
