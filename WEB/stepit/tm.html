Ghidul studentului pentru invatarea HTML

HTML este limbajul de baza pentru crearea paginilor web.
El permite structurarea informatiei astfel incat aceasta sa fie afisata corect in browser.


De ce invatam HTML?

  • Pentru a crea pagini web simple
  • Pentru a intelege structura unui site.
  • Pentru a lucra mai usor cu CSS si JavaScript.
  • Pentru a construi proiecte personale.

    • Notiuni importante:

    1. Titluri
    2. Paragrafe
    3. Liste
    4. Linkuri
    5. Imagini
    6. Formulare

    Atentie: HTML nu este un limbaj de programare, ci un limbaj de marcare.
